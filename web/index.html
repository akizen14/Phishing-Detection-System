How It Works

Hybrid Detection System

This platform uses a two-phase detection approach to identify phishing threats:
  • Phase 1 - Signature Detection: Instant verification against PhishTank's verified phishing database
  • Phase 2 - Structural Analysis: NCD-based DOM structure comparison for unknown URLs

Detection Sources:
  • PhishTank Database - Community-verified phishing reports with high confidence
  • NCD Analysis - Normalized Compression Distance algorithm for structural similarity
  • OSINT Intelligence - Domain, SSL, and hosting information for comprehensive analysis
